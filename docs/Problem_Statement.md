Problem:

FactoryOps AI is a Multi-Agent Manufacturing COO built with Google Agent Development Kit (ADK). The system coordinates specialized AI agents responsible for production, inventory, and quality analysis. These agents use reusable manufacturing tools and skills to assess factory health. The same manufacturing capabilities are exposed through an MCP server, enabling interoperability with external AI systems. An Executive Decision Engine evaluates overall business risk, recommends executive actions, and supports human approval for high-impact decisions.

Expected Benefits:
Faster operational decision-making

Reduced manual analysis

Unified operational visibility

Executive decision support with human approval

Interoperability through Model Context Protocol (MCP)

Demonstration of enterprise AI architecture using Google ADK
